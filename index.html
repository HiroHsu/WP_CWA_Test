WordPress 外掛安裝說明

您已成功將專案轉換為 WordPress 外掛架構。

1. 檔案位置

請將 taiwan-weather-warnings.php 和 css/ 資料夾放入 WordPress 的 wp-content/plugins/taiwan-weather-warnings/ 目錄中。

2. 啟用外掛

登入 WordPress 後台，前往「外掛」列表，啟用 Taiwan Weather Warnings。

3. 設定 API Key

啟用後，前往「設定」 > 「台灣氣象警示」，輸入您的 CWA API Key。

4. 顯示警報

在任何文章或頁面中使用 Shortcode：
[taiwan_weather_warnings]
